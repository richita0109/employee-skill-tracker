# Employee Skill Tracker

## Features
- Add, Edit, Delete Employees
- Skill Tracking System
- Search Employees
- Analytics Dashboard (Charts)

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB

## How to Run Locally

1. Start MongoDB

2. Run backend:
   cd server
   npm install
   npm start

3. Run frontend:
   npm install
   npm run dev

## Notes
- Uses local MongoDB (not deployed yet)
